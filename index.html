CloudForm develops interactive web applications and social games. Our goal is to create engaging digital experiences that connect users through shared content.

Guess My Like

Guess My Like is a social game developed by CloudForm where players try to guess who among them liked a specific YouTube video or Spotify track.

To enable the game experience, users may sign in with Google so the application can securely access only the data necessary for its core functionality.

Google User Data

CloudForm applications use Google OAuth to request access only to the information required to provide their intended features. Google user data is used solely for app functionality and is not sold or shared with third parties.
