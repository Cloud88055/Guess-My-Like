Privacy Policy - Guess My Like

Effective date: 2026

This Privacy Policy describes how Guess My Like ("the App"), operated by CloudForm ("we", "our", or "us"), collects, uses, and protects user information.

Information We Collect

When users sign in with Google, we may access limited information such as:

  • Name and email address
  • Public YouTube data (videos, playlists, and channel information)

This data is accessed only with the user's explicit consent via Google OAuth.

How We Use Information

  • To authenticate users securely
  • To provide the core functionality of the application
  • To generate content suggestions within the game
  • To improve user experience

Google User Data

Guess My Like accesses Google user data only as necessary to provide its core functionality.

  • We do not sell Google user data
  • We do not share Google user data with third parties
  • We do not use Google user data for advertising purposes

Data Sharing

We do not sell or share personal data with third parties.

Data Security

We implement reasonable security measures to protect user data.

Data Retention and Deletion

We do not store Google user data permanently on our servers.

Any data accessed via Google APIs is processed in real-time and is not retained after the user session ends.

Temporary data may be stored only during the active session and is automatically deleted when the session ends or the application is closed.

Users can revoke access at any time via their Google account settings:
https://myaccount.google.com/permissions

Once access is revoked, the application will no longer have access to any Google user data.

User Control

Users can revoke access at any time via their Google account settings.
